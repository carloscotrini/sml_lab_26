Statistical significance vs effect size

With large n, even trivial effects become “statistically significant.” Is this a seat number?

A p-value asks: is the estimated effect distinguishable from zero? It does not ask: is the effect big enough to care about? As the sample size grows, the standard error shrinks and the t-statistic for any non-zero effect — no matter how meaningless — eventually crosses any fixed significance threshold:

\[ t = \hat\beta / \mathrm{SE}(\hat\beta), \qquad |t| \to \infty \text{ as } n \to \infty \text{ for any } \beta \neq 0. \]

To see this, imagine a variable you know is meaningless — the seat number assigned to each student on exam day, \(S \sim \mathrm{Uniform}(1, 200)\). Suppose the seat has a vanishingly tiny true effect on the score: 0.003 points per seat, which amounts to a 0.6 point swing across the whole room on a 100-point exam. No educator alive would act on an effect that small. Yet when we include \(S\) in a regression and crank up the sample size, its p-value will eventually plummet below 0.001. The coefficient stays tiny. The effect stays meaningless. Only the significance stars multiply.

Catchphrase for the semester: whenever a tiny coefficient attains dazzling significance, ask — “Is this a seat number?”

1. The seat-number regression

To make the point concrete, we add a seat number to the usual exam DGP. Imagine the exam hall has slightly better lighting or ventilation in the back rows — a tiny, nearly imperceptible effect of \(\beta_S = 0.003\) points per seat. Over the whole room that is a 0.6-point swing on a 100-point exam: economically, it might as well be zero. The DGP now reads

\[ Y \;=\; 50 \;+\; 8\log X_1 \;+\; 3 X_2 \;+\; 6U \;+\; 0.003\, S \;+\; \varepsilon, \qquad S \sim \mathrm{Uniform}(1,200). \]
